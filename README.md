This project demonstrate the issue with datastore preferences on desktop throwing a "java.lang.ClassNotFoundException: sun.misc.Unsafe".

To trigger the issue run the app as a distributable, either using the "runDistributable" gradle task or by launching the .app generated by the "createDistributable" task.
